# DriveOutApp Netlify v1

Frontend:
- index-v1.html
- css/styles-v1.css
- js/api-v1.js
- js/app-v1.js

Backend:
- apps-script/Code-v1.gs

Deployment:
1. Deploy Code-v1.gs as Google Apps Script Web App.
2. Copy Web App URL into js/api-v1.js.
3. Rename index-v1.html to index.html for Netlify, or configure publish accordingly.
4. Drag folder to Netlify.
